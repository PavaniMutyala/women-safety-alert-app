# AI-Powered Women Safety Alert App

## Problem Statement
AI for Communities, Access & Public Impact

## Project Overview
An AI-based Women Safety Alert System that provides:
- One-tap SOS
- AI voice distress detection
- Real-time location tracking
- Automatic alert to contacts & authorities
- Safe route prediction

## Tech Stack
- Frontend: React Native / Flutter
- Backend: Node.js + Express
- Database: MongoDB
- AI: Speech Recognition + Risk Analysis
- Hosting: AWS / Firebase
